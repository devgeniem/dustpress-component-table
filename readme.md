# Installation

We recommend that you install DustPress component with Composer, but it is also possible to do it manually.

# Composer

```$ git clone devgeniem/dustpress-component-boilerplate```

Or add it into your composer.json

```
{
	"require": {
		"devgeniem/dustpress": "*"
	}
}
```
